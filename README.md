# Bulding-managment-system-
This website is used for tracking record of maintenance collection and Monthly Expense of a building <br>
This app provide features like <br>1) Member Management for Add New Member <br>2) Monthly Expense Management for showing Expense with Category-wise Expense Breakdown
 <br>3) Recent Payment History <br>4) Admin Settings for providing power to Admin for<br> a) Changing Username & Password of Admin <br> b) Modifying Monthly Dues<br> c) Record Advance Payment<br> d) View Audit Log<br> e) Reset All Data <br>5) Due Change History <br>6) Advance Balance Summary<br>7) Community Feedback & Suggestions with voting system
